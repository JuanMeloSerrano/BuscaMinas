El juego lo cree con el framework Vue.js y css grid.
Tambien utilicé tegnología webpack para la gestión de de dependencias, con el cual
comprimí todo el juego en una carpeta llamada dist, la que contiene el archivo index y un 
archivo bundle.js que es donde está todo el codigo comprimido. Es decir, que con los archivos de
la carpeta dist el juego funciona.
La carpeta "buscaminas" contiene todos los archivos separados y 
tambíen la anterir carpeta mencionada "dist".
